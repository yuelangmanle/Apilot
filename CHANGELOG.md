# 更新日志

## v1.8.0 (2026-05-29)

### 重大修复
- **修复模板添加不生效** - 从模板创建API时，数据现在正确写入数据库（之前走进了更新逻辑，数据库里没有这条记录导致保存无效）
- **修复请求体JSON未解析** - 测试API时，JSON请求体现在正确解析并发送（之前发送的是空对象）
- **设置持久化** - 暗黑模式、自动发现等设置现在重启后保留

### 改进
- API Key输入改为密码模式，支持眼睛图标切换显示/隐藏
- 模型列表为空时不再崩溃，支持手动输入模型名称
- 表单验证增加null安全处理
- 数据库读取增加防御性错误处理
