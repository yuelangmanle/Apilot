# 更新日志

## v1.9.0 (2026-05-29)

### 新增
- **macOS桌面端支持** - 新增macOS安装包（DMG格式）
- **自适应布局** - 桌面端使用NavigationRail侧边导航，手机端使用底部导航栏
- **宽屏居中** - 列表和表单在宽屏设备上自动居中显示

### 修复
- API Key输入框旁的按钮改为粘贴功能（之前是复制，对新建无用）
- API地址输入框新增粘贴按钮
- 移除列表页面重复的设置/同步入口（已在底部导航栏中）
