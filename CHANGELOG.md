# 更新日志

## v1.13.0 (2026-05-29)

### 修复
- **移除不稳定的QR扫描** - mobile_scanner在部分Android设备上崩溃，改为手动输入IP方式
- **同步改为纯HTTP直连** - 去掉UDP多播依赖，两台设备通过IP直连，更可靠
- **简化同步流程** - 显示本机IP和QR码 → 对方手动输入IP连接 → 互相收发配置

### 使用方法
1. 设备A：同步页面 → 我的二维码 → 记下IP地址
2. 设备B：同步页面 → 手动连接 → 输入设备A的IP → 连接
3. 连接成功后可发送/接收/双向同步配置
